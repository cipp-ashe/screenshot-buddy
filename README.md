
# Screenshot Task Manager

A React-based web application that helps you convert screenshots into actionable tasks. Built with modern web technologies and AI capabilities.

## Features

- Upload and analyze screenshots to automatically generate tasks
- AI-powered task title and description generation (optional)
- Organize tasks with tags and status updates
- Local storage persistence for task management
- Export functionality for task backup
- Responsive design for all device sizes

## Tech Stack

- [React](https://reactjs.org/) - UI Framework
- [TypeScript](https://www.typescriptlang.org/) - Type Safety
- [Vite](https://vitejs.dev/) - Build Tool
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [shadcn/ui](https://ui.shadcn.com/) - UI Components
- [Google Gemini](https://ai.google.dev/) - AI Integration (Optional)

## Getting Started

1. Clone the repository
```sh
git clone <repository-url>
cd screenshot-task-manager
```

2. Install dependencies
```sh
npm install
```

3. Start the development server
```sh
npm run dev
```

4. (Optional) To enable AI features:
- Get a [Gemini API key](https://makersuite.google.com/app/apikey)
- Add your API key in the application settings

## Usage

1. Upload screenshots (JPEG, PNG, GIF, or WebP format, up to 5MB)
2. Edit task details, status, and add tags to organize your work
3. Tasks are automatically saved in your browser's local storage
4. Use the export feature to backup your tasks
